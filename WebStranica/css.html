CSS

CSS je kratica od (eng.) Cascading Style Sheets. Radi se stilskom jeziku, koji se rabi za opis prezentacije dokumenta napisanog pomoću markup (HTML) jezika.


CSS možemo pisati kao stilove u zaglavlju HTML dokumenta (tj. između <style> i </style> elementa)

<style type="text/css">
h1 { color: blue }
</style>
CSS možemo pisati i unutar samih HTML tagova

<p style="color: magenta;">Neki tekst</p>
CSS možemo definirati u posebnom dokumentu i koristiti pomoću poziva

<link rel="stylesheet" href="xyz.css" type="text/css">
ili
<style type="text/css">
@import url(http://www.neki.url/neki_stil);
</style>

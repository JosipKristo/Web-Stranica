Bootstrap

Bootstrap je najpopularniji html, css, javascript framework koji se koristi kao baza za kreiranje responzivnih web stranica i web aplikacija.
Poznat je još i pod imenom Twitter Bootstrap

.
Sadrži CSS i HTML za tipografiju, ikone, oblike, dugmiće, tabele, rešetkasti (grid) prikaz, navigaciju, posebno napravljene jQuery dodatke i podršku za responzivni prikaz stranice (Prikaz na različitim velićinama uređaja).
Rešetkasti prikaz se sastoji od 12 kolona u jednom redu. To znači da je moguće stranicu podijeliti u 12 različitih kolona ili naprimjer na jednu kolonu koja zauzima prostor veličine 3 kolone plus druga kolona koja zauzima prostor 9 kolona.
Detaljnije informacije o Bootstrap frameworku možete pogledati na stranici http://getbootstrap.com/
